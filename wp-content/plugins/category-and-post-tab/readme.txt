This plugin allows to create wordpress tab widget and shortcode using ajax load more functionality.

== Description ==

Category and post wordpress tab is the powerful plugin for the Home and Sub-Pages to create tab content block and widget which will make better experience to display the categories into tab and loads post according to selected category tab.

Allows to create unlimited shortcodes and widget to view in the same or any sidebar or content page.

This plugin also supports woocommerce tab to load category and products with ajax paging load more option.

= Features =

* Supports ajax to view next posts of the selected category tab by ajax load more functionality.
* Responsive design and easy to customize template.
* Four new clean and modern designs.
* Supports unlimited widget on same or any pages.
* Allows to modify colors of text and background for widget title, post title or category tab.
* Allows to select limited categories from admin panel.

== Installation ==

1. Upload and extract the zip file downloaded to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'CPT' page from dashboard and create new shortcode to add into any page content (pro version only)
4. Configure widgets from Appearance > Widgets and assign to page sidebar.

== Frequently asked questions ==
